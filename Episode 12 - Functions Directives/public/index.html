Directives

Definisi :

Merupakan sebuah keyword didalam tailwind css yang bisa kita gunakan yang format penulisannya adalah "at-rules", atau kombinasi antara simbol @ dan rules nya apa. Yang kemudian bisa kita gunakan didalam css yang akan menawarkan fungi fungsi spesial terhadap proyek yang sedang kita kerjakan.

Directives

@tailwind :

adalah ketika kita ingin melakukan generate terhadap file css nya. Secara default terbagi menjadi 3 (base, component, dan utilities).

base : merupakan style-style utama yang menjadi dasar dari tailwind, didalamnya terdapat seperti reset, ukuran/style dasar dari semua html.
component : tailwind akan menambahkan kelas kelas komponen kalau ada, atau kelas kelas komponen yang didaftarkan oleh plugin.
utilities : tailwind akan menambahkan kelas kelas utility yang akan kita gunakan.
variants (non-default) : contoh sederhananya seperti hover, focus, active, dan sebagainya.

Directives

@layer :

biasa digunakan ketika kita ingin memodifikasi isi dari 3 directives diatas (maksudnya yang base, component, dan utilities). Misal sederhananya kita ingin mengatur agar secara default element h1 berukuran 2xl, maka hal seperti itu bisa diatur melalui directives yang satu ini.

Directives

@apply :

digunakan untuk melakukan penyimpanan utility-utility class yang sudah ada di tailwind untuk kemudian dimasukan kedalam custom css kita secara inline.
contohnya : ingin menambahkan kelas custom, tapi property nya memakai property milik tailwind.
